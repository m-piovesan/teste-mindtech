## Instruções para rodar

### Back-end

Primeiro, instalar as dependências:

```bash
npm install
```

Após isso, iniciar o servidor com:

```bash
npm run start
```

### Front-end

Após inicializar o back-end, instalar as dependências do front-end:

```bash
npm install
```

Por fim, iniciar a aplicação com:

```bash
npm run dev
```
